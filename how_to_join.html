Bedrock Edition

Aktuelle Server Version: v1.18.31 ¹

Im Folgenden ist eine Anleitung zu sehen, wie ihr den Server in der Bedrock Edition von Minecraft hinzufügt. Diese umfasst die Versionen für Android, iOS, Nintendo Switch, PlayStation, XBox und weitere Konsolen. Spieler der Bedrock Edition benötigen KEINEN Java Account und sind im Spiel mit einem Punkt vor der Namen gekennzeichnet. ²

¹) Hierbei muss auf der angegebenen Version gespielt werden, da andernfalls kein Beitritt auf den Server möglich ist. Sollte die angegebene Version veraltet sein UND kein Beitritt möglich sein, so wendet euch gerne im Discord an einen Teamer!
²) Mit einem Punkt gekennzeichnete Spieler spielen mit einem registrierten und legitimierten Account von Microsoft. Der Punkt verhindert, dass der Name einem aus der Java Edition nachgeahmt werden kann und umgekehrt. Als UUID wird die von Microsoft an die Bedrock Edition übergebene UUID verwendet.
