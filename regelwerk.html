Regelwerk

  1. Kommunikation

    2. A ) Kommunikation, sowohl in schriftlicher als auch mündlicher Form ist ausschließlich ohne Beleidigungen, Mobbing, nicht einvernehmlicher Provokation sowie ohne Verwendung von rechtswidrigen Zeichen und Schriften zulässig. ¹

    B ) Der Umgang untereinander sollte stets respektvoll und höflich sein, nur so macht das Spielen für alle gleichermaßen Spaß! ¹

    C ) Spam in jedweder Form ist verboten. ¹

    D ) CAPS ist verboten. ¹

    E ) Werbung ohne ein gültiges Zertifikat über die Werbeerlaubnis seitens des Servers ist verboten. ⁴

  2. Bauwerke

    3. A ) Bauwerke dürfen nicht rechtswidrig sein. ³ ⁴

    B ) Bauwerke dürfen Redstone Schaltungen beinhalten, jedoch keine Clocks, wenn sie den Server zum Laggen bringen können. ³

    C ) Clocks müssen ausschaltbar sein und nach jedem Verlassen des Server dementsprechend deaktiviert werden. ¹ ³

    D ) Es ist gestattet, Fallen zu errichten. Jedoch sind diese sichtbar zu markieren und/oder die Spieler darauf aufmerksam zu machen, sollten sie den Standort besuchen. Sollte es an dem Standort einen Warp-Punkt geben, ist in einem Umkreis von 7 Blöcken sichtbar ein Hinweis anzubringen. ³ ⁴

  3. Mods/Clients/Interaktion

    4. A ) Die Verwendung von anderen oder modifizierten Clients, als den Standard Vanilla/Bedrock Client, ist mit dem Team vor Gebrauch abzusprechen und ist andererseits untersagt. ²

    B ) Die Verwendung von Mods, die Einfluss auf das Spiel haben oder haben könnten, sind untersagt. Ein Beispiel dafür ist die Fishing Mod / Auto Fishing. ³

    C ) Das AFK-Cobblen ist verboten. ³

    D ) Das PlayTime-Farmen ist verboten. Zum Beispiel AFK-Brunnen. ³

    E ) Absichtliches BugUsing ist verboten. ¹ Außerdem ist das absichtliche Verschweigen eines Bugs verboten. ¹ ³

  4. Skins

    5. A ) Die Verwendung von rechtswidrigen oder provokativen Skins ist verboten. ⁴

  5. Konten

    6. A ) Jedem Spieler ist es getattet, jeweils einen Java- und einen Bedrock Account auf dem Server zu besitzen...

    B ) .. jeder weitere Account je Edition gilt als MultiAccounting und das ist untersagt. ²

  6. Spielernamen / Nicknamen

    7. A ) Spieler- oder Nicknamen gelten als schriftliche Kommunikation und dürfen nicht gegen P.1,A.A verstoßen. ¹ ²

  7. Handel

    8. A ) Die Preisgestaltung bei privatem Handel obliegt den Spielern selbst.

    B ) Betrug ist verboten. ³ ⁴

    C ) Die kostenpflichtige- oder freie Weitergabe von durch /enchant oder /fix modifizierten Items ist verboten. ³ ⁴

  8. Widerruf einer Ahndung / Einspruch

    9. A ) Ein Spieler, dem vorgeworfen wird, einen Verstoß begangen zu haben, hat das Recht dazu, sich zur Sache zu äußern und sich zu verteidigen.

    B ) Gegen ein Urteil kann innerhalb von 7 Tagen Revision eingelegt werden. Es ist je Urteil nur eine Revision möglich.

Ahndungsanordnung

¹ ) Bei erstmaligem Verstoß drohen eine oder mehrere der Strafen (Verwarnung, Kick, Temporärer Mute, Temporärer Freeze). Bei wiederholtem Vergehen droht ein Bann von bis zu 7 Tagen. Optional kann es zu einem permanenten Mute oder Freeze kommen.

² ) Bei erstmaligem Verstoß droht ein Kick oder ein Bann von bis zu 30 Tagen. Bei wiederholtem Verstoß droht ein Bann von bis zu 180 Tagen. In schweren Fällen kann es zu einem permanenten Bann kommen.

³ ) Bei erstmaligem Verstoß droht ein Kick oder ein Bann von bis zu 24 Stunden und zuzüglich eine der folgenden Strafen (50% Taler-Reset, 50% Spielzeit-Reset, Kisten- und Inventar-Reset), sowie, sofern zutreffend, Abriss des Bauwerkes ohne Erstattung der Ressourcen. Bei wiederholtem Verstoß drohen eine oder mehrere der Strafen (Taler-Reset, Spielzeit-Reset, Insel-Reset) und optional ein Bann von bis zu 180 Tagen.

⁴ ) Bei erstmaligem Verstoß kommt es zu einem Bann von bis zu 180 Tagen. Bei wiederholtem Verstoß kommt es zum permanenten Bann.


Die Ahndungsanordnung hier ist lediglich ein Richtwert. Die tatsächliche Umsetzung kann leichter oder auch schwerwiegender ausfallen und wird individuell nach Situation und Schwere des Vergehens entschieden.
